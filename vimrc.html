Plugin management

At startup, plugins are automatically installed and the plugin manager (vim-plug) is updated. Commands are:

  • :PlugUpdate to update the plugins
  • :PlugClean to remove unused plugins

NERDTree

<Ctrl-t> or :NERDTree to open a window with directory tree

  • Open a tab with t
  • fold/unfold with Space or the mouse
  • C to change tree root to the selected node
  • m to open menu for file manipulation
  • r to refresh

ipython

Open a terminal with ipython using :IPython

At the first execution, one must select the terminal executing ipython.

  • Press <Ctrl-p> to execute a cell and jump to the next
  • :IPythonCellRun executes the whole script
  • all commands are described here

GIT

Git commands are available with :Git

Completion and help

  • Completion with tab
  • help with <Ctrl-Space>

Tags

Open a tag navigator with <Ctrl-y>

Search

:FZF to search for a file by filename

Diff mode

  • <Ctrl-↑>, <Ctrl-↓> to go to previous/next diff
  • <Ctrl-←>, <Ctrl-→> to copy diff from right to left or the reverse
  • <Ctrl-PageUp>, <Ctrl-PageDown> to take or not into account spaces in diff

Other config

  • <Space> to fold/unfold
  • <Ctrl-b> to open a terminal in the current window
  • <Ctrl-n> to open a new tab with a terminal
  • <Ctrl-w>N to be able to scroll and/or copy, i to return
  • Selection
    • Select text by v (normal), V (whole lines) or  <Ctrl-V> (columns)
    • Cut with d, copy with y, paste before with P, paste after with p
    • Put in upper case with U, in lower case with u
  • Undo/redo
    • undo with u, undo the line with U, redo with <Ctrl-r>
    • get the list of undo by :undolist
  • :cq to exit with error (to abort a git commit)
